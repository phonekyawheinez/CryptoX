A comprehensive suite of security utilities for hashing, key generation, and encryption. 
This toolkit is designed for developers and security enthusiasts
to perform common cryptographic tasks through a simple,modular interface.

Features
1. Multi-Algorithm Hashing
String Hashing: Generate MD5, SHA-1, SHA-256, and SHA-512 hashes instantly.

File Hashing: Calculate checksums for local files to verify integrity.

Hash Comparison: A dedicated "Hash Check" tool to compare two files and ensure they are identical (bit-for-bit).

2. Entropy-Based Password Generator
Uses a multi-factor seed approach for increased randomness.

Generates passwords using a combination of:

System Time: High-resolution timestamps.

Process ID (PID): Environment-specific entropy.

Random Library: Cryptographically secure pseudo-random numbers.

3. Asymmetric Key Generation
Supports Public/Private key pair generation for multiple algorithms (RSA, DSA, etc.).

Exports keys in standard formats for use in secure communications.

4. Legacy Wireless Encryption
WEP Encryption/Decryption: Tools to encrypt and decrypt data using the WEP (Wired Equivalent Privacy) protocol for educational or recovery purposes.

Security Disclaimer
This toolkit is provided for educational and administrative purposes only.

WEP is a deprecated protocol and is not secure for modern data protection.

Always ensure you are following best practices when handling private keys.
